# todo-firebase

A simple todo app, built on [Firebase](https://console.firebase.google.com/)

# Prerequisites

* Node
* [Firebase CLI](https://firebase.google.com/docs/emulator-suite)
* Yarn

# Debug on local

In one terminal, you should run:
```sh
yarn emulator
```

And in another, run:
```sh
yarn start
```

# Deploy
```sh
yarn deploy
```
